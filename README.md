# tanX
A 2D tank game written in c++14 using SDL

## Library
+ SDL 1.2
+ SDL_ttf 2.0
+ SDL_mixer 1.2

## How to Play
build this project to get server and client

start the server (only one player needs to do this)

`./lnx number_of_players`

start the client

`./tanX server_ip`

use arrow keys to move, spacebar to shoot
